# 每金 - 設計理念與規劃

## 選定設計方向：溫暖極簡主義 × 互動詩意

### 設計哲學
**核心概念**：在寧靜的空間中，用簡潔的排版和溫暖的色彩，讓每一句金句成為視覺焦點。設計不搶戲，而是襯托內容。

### 設計運動
- **視覺基礎**：極簡主義 + 日式美學（留白、寧靜、專注）
- **情感基調**：溫暖、可親、略帶幽默感
- **交互理念**：輕微的動畫反饋，讓使用者感受到每次互動都是有意義的

### 核心設計原則

1. **內容至上**：金句是唯一的主角，所有視覺元素都應該為其服務
2. **留白的力量**：大量的負空間，讓眼睛有呼吸的空間
3. **溫暖親近**：湖水藍色傳達寧靜與信任，白色與灰色保持專業感
4. **微妙互動**：點擊、按空白鍵時有輕微的動畫反饋，但不過度

### 色彩哲學

| 顏色 | 用途 | 心理效應 |
|------|------|--------|
| 湖水藍 (#4A90E2 或 #5BA3D0) | 主要品牌色、按鈕、強調 | 寧靜、信任、清晰思考 |
| 白色 (#FFFFFF) | 背景、卡片 | 純淨、開放、呼吸感 |
| 專業灰 (#6B7280 或 #9CA3AF) | 文字、分隔線、輔助 | 平衡、專業、易讀 |
| 淡灰 (#F3F4F6) | 次要背景、懸停狀態 | 溫和、柔和 |

### 版面設計範式

- **中心焦點式**：金句卡片居中，上下有充足的負空間
- **垂直流動**：從上到下的自然閱讀順序
- **非對稱細節**：側邊可加入微妙的裝飾元素（如小圖標、引號），但不破壞中心焦點

### 簽名視覺元素

1. **金句卡片**：圓角卡片（border-radius: 12px），輕微陰影，內部有充足的內邊距
2. **互動提示**：「點擊或按空白鍵」的文案，用灰色、小字體、淡出效果
3. **類別標籤**：小型、圓形、湖水藍背景、白字，位於卡片上方或下方

### 交互動畫哲學

- **點擊反饋**：卡片輕微縮放 (scale: 0.98) + 淡出舊金句、淡入新金句（300ms）
- **按鍵反饋**：空白鍵按下時，卡片有輕微的上下彈動（bounce）
- **懸停效果**：卡片邊框或陰影加深，提示可點擊
- **頁面進入**：卡片從下方淡入滑上（300ms）

### 字體系統

- **標題/品牌**：Noto Serif TC（思源宋體）- 優雅、傳統感，用於「每金」標題
- **金句內容**：Noto Sans TC（思源黑體）- 清晰易讀，用於金句主文本
- **輔助文案**：Noto Sans TC 較小字重 - 用於類別、提示文本

### 排版層級

| 層級 | 用途 | 字體 | 大小 | 字重 |
|------|------|------|------|------|
| H1 | 「每金」品牌 | Noto Serif TC | 48px | 700 |
| H2 | 金句主文本 | Noto Sans TC | 28-32px | 400 |
| Body | 類別標籤 | Noto Sans TC | 14px | 500 |
| Small | 互動提示 | Noto Sans TC | 12px | 400 |

## 金句資料庫規劃

### 資料結構

```javascript
{
  id: number,
  text: string,        // 金句內容（≤50字）
  category: string,    // 'humor' | 'serious' | 'lunyu' | 'zhuangzi' | 'laozi'
  explanation: string, // 白話文解釋（如為古文）
  author?: string      // 作者（可選）
}
```

### 類別定義

| 類別 | 英文 | 特徵 | 示例 |
|------|------|------|------|
| 幽默 | humor | 輕鬆、逗趣、帶點自嘲 | 人生如戲，全靠演技。 |
| 正經 | serious | 勵志、深刻、發人深省 | 每一次跌倒，都是為了學會更好地站起來。 |
| 論語 | lunyu | 孔子及弟子言論 | 論語原文 + 白話解釋 |
| 莊子 | zhuangzi | 莊周及道家思想 | 莊子原文 + 白話解釋 |
| 老子 | laozi | 老子《道德經》 | 老子原文 + 白話解釋 |

### 初始金句集合

#### 幽默類（humor）

1. 人生如戲，全靠演技。
2. 我不是在浪費時間，我是在思考人生。
3. 別人的成功故事，都是別人的故事。
4. 失敗乃成功之母，但我媽沒生我這麼多次。
5. 人生就像騎自行車，為了保持平衡，你必須不斷前進。

#### 正經類（serious）

1. 每一次跌倒，都是為了學會更好地站起來。
2. 改變不了過去，但能塑造未來。
3. 你的努力，會在某個時刻閃閃發光。
4. 不是所有的堅持都有結果，但放棄一定沒有。
5. 生活的意義，在於你如何度過每一個平凡的日子。

#### 論語類（lunyu）

1. 原文：「學而時習之，不亦說乎？」
   解釋：學習知識後，經常溫習練習，不是很快樂嗎？

2. 原文：「知之為知之，不知為不知，是知也。」
   解釋：知道就是知道，不知道就是不知道，這才是真正的智慧。

3. 原文：「己所不欲，勿施於人。」
   解釋：自己不想要的東西，不要強加給別人。

4. 原文：「三人行，必有我師焉。」
   解釋：三個人一起走路，其中必然有我的老師。

5. 原文：「發奮圖強，不舍晝夜。」
   解釋：要努力向上，不分白天黑夜地去追求進步。

#### 莊子類（zhuangzi）

1. 原文：「逍遙遊」- 無為而無不為
   解釋：不刻意去做，反而一切都能自然完成。

2. 原文：「相濡以沫，不如相忘於江湖。」
   解釋：互相幫助度過困難，不如各自獨立地生活。

3. 原文：「至人無己，神人無功，聖人無名。」
   解釋：最高境界的人沒有自我執著，神仙般的人不計功績，聖人不追求名聲。

4. 原文：「庖丁解牛」
   解釋：做事要找到規律和竅門，才能事半功倍。

5. 原文：「夢蝶之喻」
   解釋：莊周夢見自己變成蝴蝶，醒後不知是莊周夢蝶，還是蝴蝶夢莊周。

#### 老子類（laozi）

1. 原文：「道可道，非常道。名可名，非常名。」
   解釋：能用言語說出來的道理，就不是永恆不變的道。

2. 原文：「無為而無不為。」
   解釋：不刻意去做，反而能成就一切。

3. 原文：「上善若水，水善利萬物而不爭。」
   解釋：最高的善就像水一樣，滋潤萬物卻不與之爭奪。

4. 原文：「知人者智，自知者明。」
   解釋：了解別人是聰慧，了解自己才是真正的聰慧。

5. 原文：「禍兮福之所倚，福兮禍之所伏。」
   解釋：禍與福相互依存，禍中可能隱藏著福，福中也可能隱藏著禍。

## 技術實現規劃

### 前端架構

- **框架**：React 19 + Tailwind CSS 4
- **狀態管理**：useState（簡單的金句輪換）
- **動畫**：Framer Motion（淡入淡出、彈動效果）
- **組件庫**：shadcn/ui（按鈕、卡片等基礎組件）

### 核心功能實現

1. **金句輪換**：
   - 點擊卡片 → 隨機選取新金句
   - 按空白鍵 → 隨機選取新金句
   - 淡出舊金句，淡入新金句

2. **類別顯示**：
   - 卡片上方顯示類別標籤
   - 類別用湖水藍背景、白字

3. **解釋顯示**（古文類別）：
   - 金句下方顯示白話文解釋
   - 字體較小、顏色較淡

4. **響應式設計**：
   - 手機端：卡片寬度 90vw，金句字體 20px
   - 平板端：卡片寬度 70vw，金句字體 24px
   - 桌面端：卡片寬度 600px，金句字體 28px

### 檔案結構

```
client/src/
├── pages/
│   └── Home.tsx          # 主頁面
├── components/
│   ├── QuoteCard.tsx     # 金句卡片組件
│   └── CategoryBadge.tsx # 類別標籤組件
├── lib/
│   └── quotes.ts         # 金句資料庫
├── hooks/
│   └── useRandomQuote.ts # 隨機金句 hook
└── index.css             # 全局樣式
```
